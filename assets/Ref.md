How did using Figma designs as references affect your coding process?
Using the Figma design as a reference helped me create a wireframe and better understand the layout of the project. It gave me a clear idea of where each element should be placed, which made the development process more organized.



What challenges did you encounter when aligning your code with the design specifications?

One challenge I encountered while aligning my code with the design specifications was creating the shadow effect for the blog section. This was a new concept for me, and I had to learn how to properly apply it using CSS. Figuring out how to place the shadow on one side and adjusting its appearance while also adding text to the section required some experimentation and practice. Through this process, I gained a better understanding of how styling properties work and how small design details can impact the overall look of a webpage.

How can the feedback and community resources on Frontend Mentor help you improve as a developer?
Feedback and community resources on Frontend Mentor helped me understand the importance of spacing, especially padding, margins, and borders around images. These elements play a key role in properly aligning images and text, which improves the overall layout during the development process.
